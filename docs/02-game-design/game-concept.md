# 游戏概念与定位

## 项目名称

电竞比赛模拟器 2（EsportManager 2）- 第二代桌面端应用

## 项目定位

基于用户定制的"年度常规赛事+跨年度顶级杯赛"体系，打造轻量化、高还原度的"自娱自乐式"电竞赛事模拟游戏。

通过自动化赛程生成、胜负判定、积分统计及可视化展示，替代手工记录，完整复现从联赛到跨年度杯赛的全流程，聚焦赛事模拟的真实性与操作便捷性。

## 项目目标

1. 支持 **4个赛区 × 14支战队** 管理
2. 实现年度常规赛事周期，及每年1次的"Super洲际年度邀请赛"
3. 自动化完成各赛事赛程生成、基于战力值和正态分布算法自动判断比赛胜负
4. 提供全赛事可视化对阵图（含抽签过程）、战队数据中心、年度积分总结
5. 最终生成年度全球前16战队排名及洲际超级杯赛冠军

## 核心特性

### 4大赛区

| 赛区 | 代码 | 地区 | 战队数量 |
|------|------|------|----------|
| LPL | LPL | 中国大陆 | 14支 |
| LCK | LCK | 韩国 | 14支 |
| LEC | LEC | 欧洲 | 14支 |
| LCS | LCS | 北美 | 14支 |

**总计**: 56支战队、280名选手

### 15个赛季阶段

完整的年度赛事周期：

```
春季赛常规赛 → 春季赛季后赛 → MSI → 马德里大师赛 →
夏季赛常规赛 → 夏季赛季后赛 → Claude洲际赛 → 世界赛 →
上海大师赛 → ICP洲际对抗赛 → Super洲际赛 → 年度颁奖 →
转会期 → 选秀(每4年) → 赛季结束
```

### 选手生命周期

- **入行**: 16-19岁，通过选秀进入职业联赛
- **成长**: 每赛季根据天赋标签提升能力值
- **巅峰**: 25-29岁，能力和稳定性达到顶峰
- **衰退**: 30岁后能力开始下降
- **退役**: 35岁以上且能力低于65时退役

### 核心玩法

1. **比赛模拟**: 基于正态分布的战力值模拟，强队有高概率获胜但允许爆冷
2. **选手培养**: 管理选手成长，平衡即战力和潜力
3. **转会市场**: AI驱动的8轮转会系统，包含自由球员、报价、竞标
4. **财政管理**: 收入支出平衡，薪资管理，避免破产
5. **荣誉积累**: 赛事冠军、MVP、年度最佳等荣誉系统

## 目标用户

- 电竞爱好者
- 足球经理类游戏玩家
- 喜欢数据模拟的用户
- 追求"自娱自乐"式模拟体验的玩家

## 技术实现

- **前端**: Vue 3 + TypeScript + Element Plus
- **后端**: Rust + Tauri 2.0
- **数据库**: SQLite
- **架构**: 桌面端应用，前后端分离
