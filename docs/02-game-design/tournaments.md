## 联赛详细规则

### 常规赛

**赛制**: BO3 双循环（每队与同赛区其他13队各交手2次）

**积分规则**:
| 结果 | 积分 |
|------|------|
| 2:0 胜 | 3分 |
| 2:1 胜 | 2分 |
| 1:2 负 | 1分 |
| 0:2 负 | 0分 |

**排名规则**: 积分相同时：
1. 历史交锋胜率
2. 总净胜小局数

### 季后赛

**晋级规则**: 常规赛前8名进入季后赛

**分组**:
- 胜者组: 第1-4名
- 败者组: 第5-8名

**赛制流程（双败BO5）**:

```
胜者组第一轮:   1 vs 4, 2 vs 3
    ↓ 胜者 → 胜者组第二轮
    ↓ 败者 → 败者组第二轮

败者组第一轮:   5 vs 8, 6 vs 7
    ↓ 胜者 → 败者组第二轮
    ↓ 败者 → 淘汰

胜者组第二轮(胜者组决赛):
    ↓ 胜者 → 总决赛
    ↓ 败者 → 败者组决赛

败者组第二轮:   败者组第一轮胜者 vs 胜者组第一轮败者
    ↓ 胜者 → 败者组第三轮

败者组第三轮:   两个败者组第二轮胜者对决
    ↓ 胜者 → 败者组决赛

败者组决赛:     胜者组决赛败者 vs 败者组第三轮胜者
    ↓ 胜者 → 总决赛

总决赛:         胜者组冠军 vs 败者组决赛胜者
```

**积分奖励**:
| 名次 | 积分 |
|------|------|
| 冠军 | 12分 |
| 亚军 | 10分 |
| 季军 | 8分 |
| 殿军 | 6分 |
| 5-8名 | 3分 |

## 国际赛事详细规则

### MSI 季中赛

**参赛队伍**: 12支（4赛区春季赛冠/亚/季军）
- 冠军 → 传奇组
- 亚军 → 挑战者组
- 季军 → 资格赛组

**赛制流程（双败BO5）**:

1. **预选赛**
   - 资格赛组: 4队两两BO5单淘汰，2胜者 → 败者组第一轮
   - 挑战者组: 4队PK，2胜者 → 败者组第二轮，2败者 → 败者组第一轮

2. **正式阶段**
   - 败者组第一轮: 资格赛胜者 vs 挑战者组败者
   - 败者组第二轮: 挑战者组胜者 vs 败者组第一轮胜者
   - 胜者组第一轮: 4传奇组对决
   - 败者组第三轮: 败者组第二轮胜者 vs 胜者组第一轮败者
   - 败者组第四轮: 2胜者对决
   - 胜者组第二轮: 2胜者对决
   - 败者组决赛: 胜者组第二轮败者 vs 败者组第四轮胜者
   - 总决赛: 胜者组冠军 vs 败者组决赛胜者

**积分奖励**:
| 名次 | 积分 |
|------|------|
| 冠军 | 20分 |
| 亚军 | 16分 |
| 季军 | 12分 |
| 殿军 | 8分 |
| 败者组第二轮败者 | 6分 |
| 败者组第一轮败者 | 4分 |

### 马德里大师赛 / Claude洲际赛

**参赛队伍**: 32支（4赛区各8队，根据常规赛排名）

**小组赛**:
- 分成8个小组（A-H），每组4队
- BO3单循环
- 积分规则: 2:0得3分, 2:1得2分, 1:2得1分, 0:2得0分
- 每组前2名晋级淘汰赛

**淘汰赛**:
- 16队分为东西半区（各8队）
- BO5单淘汰
- 轮次: 第一轮 → 半决赛 → 决赛
- 季军赛: 两个半区决赛败者
- 总决赛: 两个半区冠军

**积分奖励**:
| 名次 | 积分 |
|------|------|
| 冠军 | 20分 |
| 亚军 | 16分 |
| 季军 | 12分 |
| 殿军 | 8分 |
| 半区决赛败者 | 6分 |
| 半决赛败者 | 4分 |
| 第一轮败者 | 2分 |

### S 世界赛

**参赛队伍**: 12支（4赛区夏季赛冠/亚/季军）
- 冠军 → 直接进淘汰赛
- 亚/季军 → 小组赛

**赛制流程**:
1. **小组赛**: 8支（亚/季军）BO1瑞士轮，决出4队晋级淘汰赛
2. **淘汰赛**: 8支（4直接晋级+4小组赛晋级）单淘汰BO5
3. **季军赛**: 半决赛败者BO5决季军

**积分奖励**:
| 名次 | 积分 |
|------|------|
| 冠军 | 20分 |
| 亚军 | 16分 |
| 季军 | 12分 |
| 殿军 | 8分 |
| 淘汰赛第一轮败者 | 6分 |
| 小组赛淘汰 | 4分 |

### ICP 洲际对抗赛

**参赛队伍**: 16支（4赛区夏季赛前4名）

**分组规则**:
- 各赛区1号种子 → A组
- 各赛区2号种子 → B组
- 各赛区3号种子 → C组
- 各赛区4号种子 → D组

**小组赛**: BO3单循环，前2名获得徽章

**决赛阶段**: 根据各赛区获得徽章数排名
- 可分出前2名 → 直接决赛
- 可分出前3名 → 后两名半决赛，胜者进决赛
- 决赛: 各赛区队伍按种子对位PK（1v1, 2v2, 3v3, 4v4）

**积分奖励** (按赛区排名):
| 赛区排名 | 参赛队积分 | 未参赛队积分 |
|---------|-----------|-------------|
| 第1名 | 12分 | 6分 |
| 第2名 | 8分 | 4分 |
| 第3名 | 6分 | 3分 |
| 第4名 | 4分 | 2分 |

### Super 洲际年度邀请赛

**参赛资格**: 年度积分全球前16名

**分组**:
- 传奇组: 第1-4名（4支）
- 挑战者组: 第5-8名（4支）
- Fighter组: 第9-16名（8支）

**四阶段赛制**:

**第一阶段: Fighter组预选赛**
- 8队分A/B两组
- BO3单循环
- 每组第1名晋级，其余淘汰（各得2分）

**第二阶段: 挑战者组定位赛与晋级赛**
- 定位赛: 挑战者组两两BO5单淘汰
- 晋级赛: Fighter组晋级者 vs 定位赛败者
- 晋级赛败者淘汰（各得5分）

**第三阶段: 冠军赛预备战**
- 胜者组: 2支定位赛胜者BO5
- 败者组: 2支晋级赛胜者BO5
- 败者组决赛: 胜者组败者 vs 败者组胜者
- 败者组淘汰（各得8分）

**第四阶段: 终极冠军赛**
- 6队: 传奇组4队 + 第三阶段晋级2队
- 首轮: 传奇组3-4名 vs 晋级队
- 次轮: 首轮胜者 vs 传奇组1-2名
- 季军赛: 次轮2败者BO5
- 总决赛: 次轮2胜者BO5

**积分奖励**:
| 名次 | 积分 |
|------|------|
| 冠军 | 35分 |
| 亚军 | 30分 |
| 季军 | 25分 |
| 殿军 | 20分 |
| 第三阶段淘汰 | 8分 |
| 第二阶段淘汰 | 5分 |
| 第一阶段淘汰 | 2分 |
